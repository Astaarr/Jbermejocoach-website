## JBermejo Entrenamiento y Nutricion

Landing page creada con Next.js (App Router + TypeScript), lista para desplegar en Vercel.

### Stack

- Next.js 16
- TypeScript
- App Router
- Tailwind CSS v4

### Desarrollo local

```bash
npm install
npm run dev
```

Abrir `http://localhost:3000`.

### Build de produccion

```bash
npm run build
npm run start
```

### Variables de entorno pendientes

- `RESEND_API_KEY` (TODO: para activar el envio real del formulario)
- `NEXT_PUBLIC_SITE_URL` (recomendado para sitemap y metadatos)

### Despliegue en Vercel

1. Subir este proyecto a un repositorio Git.
2. Importar el repo en Vercel.
3. Configurar variables de entorno.
4. Deploy.
